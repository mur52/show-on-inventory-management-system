# 📦 Show On Inventory Management System

An Excel-based Inventory Management System developed for a retail clothing business to simplify inventory tracking, sales management, purchase recording, and business reporting.

## 📌 Overview

This project helps small businesses manage their daily operations using Microsoft Excel without requiring expensive software.

The workbook automatically tracks inventory movement, records purchases and sales, monitors operating expenses, and provides an interactive dashboard for business insights.
